# ✨ AI Visual Prompt Composer Pro

**A professional desktop-class web application for creating optimized AI image and video prompts.**

> Single HTML file • No backend • No framework • Works offline • Fully responsive

## 🎯 Features

### 23 Specialized Modules
1. **Style** - Visual language & aesthetics
2. **Story** - Narrative & context
3. **Subject** - Main subject definition
4. **Environment** - Surroundings
5. **Emotion** - Emotional tone
6. **Weather** - Atmospheric conditions
7. **Lighting** - Light setup
8. **Camera** - Camera parameters
9. **Composition** - Visual structure
10. **Depth** - Depth & layers
11. **Motion** - Movement & dynamics
12. **Color** - Color scheme
13. **Texture** - Surface qualities
14. **Quality** - Output quality
15. **Video** - Video parameters
16. **Negative** - Exclude elements
17. **Advanced** - Fine-tuning
18. **Output** - Export format
19. **References** - Reference images
20. **Library** - Prompt library
21. **History** - Access history
22. **Templates** - Quick templates
23. **AI Assistant** - AI enhancement

## 📖 Quick Start
1. Open `index.html` in your browser
2. Select modules from the sidebar
3. Fill in fields with options
4. Watch real-time prompt generation
5. Click "Generate" to save to history
6. Export/import projects as JSON

## 💾 Storage
- **Local Storage** - Auto-saves locally

- **No Cloud Required** - Complete privacy

## 📝 Usage

### Generate Prompt
1. Select a module from the sidebar
2. Choose options from dropdowns
3. See live preview on the right
4. Click "Generate" to add to history
5. Export as JSON or copy the prompt

### Export Project
Click the 💾 button to download your project as JSON

### Import Project
Click the 📂 button to load a previously saved project

## 🔧 Customization

Edit the `modules` array in the script to:
- Add/remove modules
- Modify field options
- Change descriptions
